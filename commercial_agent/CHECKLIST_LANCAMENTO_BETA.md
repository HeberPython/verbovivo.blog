# Checklist de Lancamento Beta

## Produto
- [ ] Nome definido.
- [ ] Promessa validada.
- [ ] PDF beta escrito.
- [ ] Templates organizados.
- [ ] Capa criada.
- [ ] Bonus definidos.
- [ ] Preco beta definido.

## Infraestrutura
- [ ] Produto cadastrado na Hotmart.
- [ ] Checkout criado.
- [ ] Link de compra testado.
- [ ] Pagina de vendas publicada.
- [ ] E-mail de suporte ativo.
- [ ] Hotmart Analytics habilitado/acessivel.
- [ ] Google Analytics ou medicao simples configurada, se usar pagina propria.

## Copy
- [ ] Headline clara.
- [ ] Prova do projeto real.
- [ ] Para quem e.
- [ ] Para quem nao e.
- [ ] Custos externos explicados.
- [ ] FAQ com objecoes.
- [ ] CTA visivel.

## Validacao
- [ ] 20 pessoas listadas para conversa direta.
- [ ] 10 mensagens enviadas.
- [ ] 5 perguntas coletadas.
- [ ] 1 post de bastidor publicado.
- [ ] 1 post explicando o fluxo de e-mail publicado.
- [ ] 1 convite para lista beta publicado.

## Vendas
- [ ] Primeiras 5 vendas monitoradas.
- [ ] Feedback pedido aos compradores.
- [ ] Objecoes anotadas.
- [ ] Pagina ajustada com base nos dados.
