# Checklist de Lancamento Beta

Produto:

**Verbo Vivo Lab: Blog Editorial Automatizado do Zero**

## Produto

- [ ] Nome final definido.
- [ ] Promessa revisada.
- [ ] PDF beta escrito.
- [ ] Roteiro do ebook seguido.
- [ ] Templates organizados.
- [ ] Capa criada.
- [ ] Bonus definidos.
- [ ] Preco beta definido em R$ 47.
- [ ] Custos externos explicados.
- [ ] Limites do produto explicados.

## Entrega

- [ ] PDF exportado e revisado.
- [ ] ZIP de templates criado.
- [ ] Pagina complementar criada.
- [ ] E-mail de boas-vindas escrito.
- [ ] Pesquisa de feedback criada.
- [ ] Arquivos sensiveis removidos.
- [ ] Nenhuma senha ou credencial exposta.

## Hotmart

- [ ] Produto cadastrado.
- [ ] Formato ebook/manual selecionado.
- [ ] Preco beta configurado.
- [ ] Garantia configurada.
- [ ] PDF enviado.
- [ ] Bonus enviados ou linkados.
- [ ] Pagina de Produto publicada.
- [ ] Checkout configurado.
- [ ] E-mail de suporte ativo.
- [ ] Link de compra testado.
- [ ] Descricao conferida.
- [ ] FAQ conferido.

## Copy

- [ ] Headline clara.
- [ ] Prova do projeto real visivel.
- [ ] Entregaveis listados.
- [ ] Para quem e.
- [ ] Para quem nao e.
- [ ] Custos externos explicados.
- [ ] Garantia explicada.
- [ ] FAQ com objecoes.
- [ ] CTA visivel no inicio e no fim.
- [ ] Sem promessa de renda garantida.

## Validacao

- [ ] 20 pessoas listadas para conversa direta.
- [ ] 10 mensagens enviadas no dia 2.
- [ ] 5 perguntas coletadas.
- [ ] 1 post de bastidor publicado.
- [ ] 1 post explicando o fluxo de e-mail publicado.
- [ ] 1 convite para lista beta publicado.
- [ ] Todas as objecoes registradas.
- [ ] Planilha de metricas preenchida diariamente.

## Vendas

- [ ] Primeiras 5 vendas monitoradas.
- [ ] Feedback pedido aos compradores.
- [ ] Objecoes anotadas.
- [ ] Pagina ajustada com base nos dados.
- [ ] Reembolsos acompanhados.
- [ ] Decisao tomada: continuar, ajustar ou pausar.
